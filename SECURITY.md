# Security Policy

## Supported versions

Security fixes are provided for the latest published release.

| Version | Supported |
| --- | --- |
| Latest release | Yes |
| Older releases | No |

## Reporting a vulnerability

Please do not open a public issue.

Use a private GitHub Security Advisory at <https://github.com/hmmhmmhm/tmux-attach-browser/security/advisories/new> or email `hmmhmmhm@naver.com`. Include the affected version, platform, reproduction steps, and potential impact.

You should receive an acknowledgment within seven days. The maintainer will coordinate validation, a fix, release timing, and public disclosure with the reporter.
